🧬 SLIIT Research Project 2025/2026

An Intelligent Assistant for Fertility Prediction, Wellness, and Support

Momera is an AI-powered IVF support platform that combines wearable health monitoring, fertility prediction, psychological support, and personalized nutrition guidance to improve patient care during the IVF journey.

Explore Project View Documents

About the Project

Infertility treatment through IVF is emotionally, physically, and financially challenging. Traditional IVF monitoring depends mainly on hospital visits and manual records, which may miss real-time changes in the patient’s health.

This project proposes an intelligent IVF assistant that uses AI, wearable IoT data, and personalized recommendations to support patients and healthcare providers.
